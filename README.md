## 电商后台管理系统(前端项目）

### 功能

> 用于管理用户账号，商品分类，商品信息，订单，数据统计等业务功能
> ![](D:\manage\src\assets\images\mall_desc01.png)

### 技术选型

#### 前端项目技术栈

- Vue
- Vue-router
- Element-UI
- Axios
- Echarts

### 项目初始化

#### 前端项目初始化步骤

1. 安装 Vue 脚手架
2. 通过 Vue-Cli 创建项目
3. 配置 Vue-router
4. 配置 Element-UI 组件库
5. 配置 Axios 库
6. 初始化 git 远程仓库
